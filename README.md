> The goal of this project is to add more features and customization to the og theme.

## 扩展内容

- 在blog底部显示所属分类

- 引入霞骛字体

- 在首页显示访问计数器

- 增加提示块短代码，包括提示`tip`、注意`note`、重要`important`、警告`warning`、危险`danger`

  用法如：``{{< admonition important "Important" >}}内容{{< /admonition >}}``
